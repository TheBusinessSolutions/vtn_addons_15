Partner(Contact) Approval

This module is made to allow you to approve or reject Partner only by the Partner approver.



Configure Users for Approval

Go to the Settings -> Users & Companies-> Users This app will add 'Can Approve Partner' group.Assign this group by activating it to that user which you want that he/she can approve Partners.
It will add Menu 'Pending Approval Contact' in Contacts and it is only visible to approver user.
When you create any new Partner it will create in Unapproved state by default and it is displayed in Ribbon. By clicking on 'Approve' ,the Partner will be moved to Contacts Menu.
Approved Partners will be visible in Contacts menu.
